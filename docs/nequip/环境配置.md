# NequIP 环境配置（详细版）

对应工程：`minimal_adl_ethene_butadiene_nequip`  
对应 conda：`ADL_NequIP`  
对应模型：`NequIP`（当前训练后端预留）

## 1. 当前阶段说明

NequIP 项目现在的目标是：

- 环境可检查
- 前置数据流程可跑通
- 训练阶段明确返回 `NotImplementedError`

也就是说：现在先把环境和前置链路打通，不是立刻完成 NequIP 训练。

## 2. 版本策略

建议：

- `Python 3.10`
- `torch` 版本按机房驱动选择
- 安装 `nequip`
- 用 `check_environment --strict` 作为最终门禁

## 3. 推荐创建流程

```bash
source ~/.bashrc
conda create -n ADL_NequIP python=3.10 -y
conda activate ADL_NequIP

conda install -y -n base conda-libmamba-solver
conda config --set solver libmamba
conda config --set channel_priority flexible

cd /share/home/Chenlehui/work/test_ADL/minimal_adl_ethene_butadiene_nequip
python -m pip install -r requirements.txt
python -m pip install nequip
```

## 7. 常见坑与处理

### 7.1 `关键环境检查未通过: nequip`

```bash
conda activate ADL_NequIP
python -m pip install nequip
```

### 7.2 训练阶段直接报 `NotImplementedError`

这不是环境损坏，是当前工程设计行为（占位提醒）。

### 7.3 `xtb` / `g16` 找不到

优先检查 PATH，不要误判为 Python 包问题。

## 8. 配置文件必须对应

`configs/base.yaml` 中：

- `training.ml_model_type: NequIP`
- `cluster.conda_env: ADL_NequIP`
