# 环境配置（ANI / MACE / NequIP）

本文档统一说明三个并列工程的 conda 环境、依赖安装与训练前自检命令。

## 1. 环境命名约定

- ANI：`ADL_env`
- MACE：`ADL_MACE`
- NequIP：`ADL_NequIP`

对应项目目录：

- ANI：`minimal_adl_ethene_butadiene`
- MACE：`minimal_adl_ethene_butadiene_mace`
- NequIP：`minimal_adl_ethene_butadiene_nequip`

## 2. 推荐创建命令（可复制）

```bash
conda create -n ADL_env python=3.10 -y
conda create -n ADL_MACE python=3.10 -y
conda create -n ADL_NequIP python=3.10 -y
```

## 3. 依赖安装建议

### 3.1 ANI 环境（ADL_env）

```bash
cd /share/home/Chenlehui/work/test_ADL/minimal_adl_ethene_butadiene
conda activate ADL_env
python -m pip install -r requirements.txt
# 按你的机房 CUDA/驱动安装 torch + torchani
```

建议至少具备：`mlatom`、`torch`、`torchani`、`pyh5md`、`xtb`。

### 3.2 MACE 环境（ADL_MACE）

```bash
cd /share/home/Chenlehui/work/test_ADL/minimal_adl_ethene_butadiene_mace
conda activate ADL_MACE
python -m pip install -r requirements.txt
python -m pip install mace-torch
```

建议至少具备：`mlatom`、`torch`、`mace` 或 `mace_torch`、`xtb`。

### 3.3 NequIP 环境（ADL_NequIP）

```bash
cd /share/home/Chenlehui/work/test_ADL/minimal_adl_ethene_butadiene_nequip
conda activate ADL_NequIP
python -m pip install -r requirements.txt
python -m pip install nequip
```

注意：当前仓库 NequIP 训练后端为预留状态，训练阶段会明确抛出 `NotImplementedError`。

## 4. 训练前统一门禁测试（强约定）

进入对应子项目目录后执行：

```bash
python scripts/check_environment.py --config configs/base.yaml --strict
python scripts/check_environment.py --config configs/base.yaml --strict --test-mlatom-xtb
```

GPU 节点可附加：

```bash
python scripts/check_environment.py --config configs/base.yaml --strict --expect-gpu
```

## 5. PBS 环境激活说明

三个 `configs/base.yaml` 已分别配置：

- ANI：`cluster.conda_env: ADL_env`
- MACE：`cluster.conda_env: ADL_MACE`
- NequIP：`cluster.conda_env: ADL_NequIP`

并且 `cluster.env_blocks` 中的 `conda activate` 已与上述环境名一致。

## 6. 关于 tmp

`tmp/` 仅用于临时脚本或中间产物，不是正式结果目录。
仓库已忽略 `/tmp/`，正式结果应保存在各子项目的 `models/`、`results/` 等标准目录。
