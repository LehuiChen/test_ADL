# MACE 环境配置

对应工程：`minimal_adl_ethene_butadiene_mace`  
对应 conda：`ADL_MACE`  
对应模型：`MACE`

## 1. 本环境负责什么

`ADL_MACE` 用于 MACE 工程的完整流程：

- 数据准备与标注阶段
- `train_main_model` / `train_aux_model`
- UQ 与选点
- `check_environment.py` 训练前门禁检查

## 2. 依赖清单（按实际检查脚本）

`scripts/check_environment.py --strict` 会重点检查：

- Python 模块：`yaml`、`mlatom`、`pyh5md`、`joblib`、`sklearn`、`torch`
- MACE 后端：`mace` 或 `mace_torch`（二选一可通过）
- 系统命令：`xtb`
- 当 `target.program=Gaussian` 时还会检查：`g16`

建议同时准备（用于分析/可视化）：

- `pandas`、`matplotlib`、`seaborn`

## 3. 创建与安装（推荐顺序）

### 3.1 新建环境

```bash
conda create -n ADL_MACE python=3.10 -y
conda activate ADL_MACE
```

### 3.2 安装项目基础依赖

```bash
cd /share/home/Chenlehui/work/test_ADL/minimal_adl_ethene_butadiene_mace
python -m pip install -r requirements.txt
```

### 3.3 安装 MACE 后端

```bash
python -m pip install mace-torch
```

若你机房镜像里叫 `mace`，也可安装 `mace`。

## 4. 系统程序检查（非 pip 包）

以下程序通常由机房环境提供：

- `xtb`
- `g16`（如果启用 Gaussian target）

检查命令：

```bash
which xtb
xtb --version
which g16
```

## 5. 训练前门禁（必须先跑）

进入 MACE 工程目录后执行：

```bash
conda activate ADL_MACE
cd /share/home/Chenlehui/work/test_ADL/minimal_adl_ethene_butadiene_mace
python scripts/check_environment.py --config configs/base.yaml --strict
python scripts/check_environment.py --config configs/base.yaml --strict --test-mlatom-xtb
```

GPU 节点建议附加：

```bash
python scripts/check_environment.py --config configs/base.yaml --strict --expect-gpu
```

建议留档：

```bash
python scripts/check_environment.py --config configs/base.yaml --strict --json-output results/check_environment_latest.json
```

## 6. 常见报错与处理

### 6.1 `关键环境检查未通过: mace_backend`

处理：

```bash
conda activate ADL_MACE
python -m pip install mace-torch
```

### 6.2 `关键环境检查未通过: xtb`

原因：`xtb` 不在 `PATH`。  
处理：在登录脚本或 PBS `env_blocks` 中补机房 `xtb` 路径。

### 6.3 `--expect-gpu` 下 CUDA 不可见

原因：未进入 GPU 节点，或 torch 与驱动不匹配。  
处理：切到 GPU 节点重测并检查 `nvidia-smi`。

### 6.4 `关键环境检查未通过: g16`

仅在 Gaussian target 开启时要求。  
处理：确认 Gaussian 环境脚本可用，`g16` 在 PATH 中。

## 7. 配置对应关系（必须一致）

`configs/base.yaml` 中应保持：

- `training.ml_model_type: MACE`
- `cluster.conda_env: ADL_MACE`
