# MACE 环境配置（分步安装 + 镜像容错）

对应工程：`minimal_adl_ethene_butadiene_mace`  
对应 conda：`ADL_MACE`  
对应模型：`MACE`

## 1. 为什么不要再用 `pip install -r requirements.txt`

你在服务器遇到的是典型外网不稳定问题：

- `Network is unreachable`
- pip 重试后卡住

所以 MACE 改成：

- 基础包走 `conda`
- 关键包走 `pip` 国内镜像
- 每一步安装完立刻验收

## 2. 先把坏环境清干净（如果之前卡死过）

```bash
conda deactivate || true
source ~/miniconda3/etc/profile.d/conda.sh
rm -rf ~/miniconda3/envs/ADL_MACE
conda create -n ADL_MACE python=3.10 -y
conda activate ADL_MACE
```

验证：

```bash
python -V
which python
python -c "import sys; print(sys.executable)"
```

## 3. 镜像连通性先测

```bash
python -m pip install -i https://pypi.tuna.tsinghua.edu.cn/simple --retries 2 --timeout 20 packaging==24.2
python -c "import packaging; print(packaging.__version__)"
```

## 4. 分步安装命令（手动版）

在 `minimal_adl_ethene_butadiene_mace` 目录执行：

### 4.1 conda 基础工具

```bash
conda install -y pip setuptools wheel
python -V
which python
```

### 4.2 conda 科学栈

```bash
conda install -y numpy pyyaml matplotlib seaborn joblib scikit-learn
python -c "import yaml,joblib,sklearn; print('base ok')"
```

### 4.3 pip(镜像) 装 `pyh5md`

```bash
python -m pip install -i https://pypi.tuna.tsinghua.edu.cn/simple --retries 2 --timeout 20 pyh5md
python -c "import pyh5md; print('pyh5md ok')"
```

### 4.4 pip(镜像) 装 `mlatom`

```bash
python -m pip install -i https://pypi.tuna.tsinghua.edu.cn/simple --retries 2 --timeout 20 mlatom
python -c "import mlatom; print('mlatom ok')"
```

### 4.5 安装 torch（按机房验证版本）

这一段必须按你机房驱动来，常见做法示例：

```bash
# 示例：按你机房可用版本安装，以下只是示例，不是唯一答案
python -m pip install -i https://pypi.tuna.tsinghua.edu.cn/simple --retries 2 --timeout 20 torch==1.12.0
```

验收：

```bash
python -c "import torch; print(torch.__version__, torch.version.cuda, torch.cuda.is_available())"
```

### 4.6 pip(镜像) 装 MACE 后端

```bash
python -m pip install -i https://pypi.tuna.tsinghua.edu.cn/simple --retries 2 --timeout 20 mace-torch
python -c "import importlib; print('mace ok', importlib.util.find_spec('mace_torch') is not None or importlib.util.find_spec('mace') is not None)"
```

如果 `mace-torch` 失败，再试：

```bash
python -m pip install -i https://pypi.tuna.tsinghua.edu.cn/simple --retries 2 --timeout 20 mace
```

## 8. 常见坑与处理

### 8.1 pip 一直 `Network is unreachable`

优先切镜像并降低重试时长，不要盲等：

```bash
python -m pip install -i https://pypi.tuna.tsinghua.edu.cn/simple --retries 2 --timeout 20 <package>
```

### 8.2 激活环境后还是 Python 2.7

这是坏环境目录残留。按第 2 节删除 `~/miniconda3/envs/ADL_MACE` 后重建。

### 8.3 `mace_backend` 检查不过

先确认 torch 正常，再装 `mace-torch`（或 fallback `mace`）。

### 8.4 `xtb` / `g16` 找不到

不是 Python 包问题，先检查 PATH 和机房环境脚本。
