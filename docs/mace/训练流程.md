# MACE 训练流程

## 1. 当前主线

MACE 现在与 ANI 对齐，主线是：

`TS.log -> prepare_ts_seed -> sample_ts_initial_conditions -> round 0 labels -> build_delta_dataset -> MACE train -> bidirectional MD -> stop on first uncertain point -> select_md_frames -> next round labels`

区别只在后端和资源：MACE 当前固定 CPU-only。

## 2. 正式第一轮入口

```bash
python scripts/run_first_round_pipeline.py \
  --config configs/base.yaml \
  --submit-mode-labels pbs \
  --submit-mode-train pbs \
  --submit-mode-md pbs
```

## 3. 最小 smoke test

```bash
python scripts/prepare_ts_seed.py --config configs/base.yaml
python scripts/sample_ts_initial_conditions.py --config configs/base.yaml --round-index 0 --num-initial-conditions 10
python scripts/run_xtb_labels.py --config configs/base.yaml --manifest results/round_000_initial_conditions_manifest.json --submit-mode pbs
python scripts/run_target_labels.py --config configs/base.yaml --manifest results/round_000_initial_conditions_manifest.json --submit-mode pbs
python scripts/update_cumulative_manifest.py --config configs/base.yaml --manifest results/round_000_initial_conditions_manifest.json
python scripts/build_delta_dataset.py --config configs/base.yaml --manifest data/processed/cumulative_labeled_manifest.json
python scripts/train_main_model.py --config configs/base.yaml --submit-mode pbs --device cpu
python scripts/train_aux_model.py --config configs/base.yaml --submit-mode pbs --device cpu
python scripts/run_md_sampling.py --config configs/base.yaml --round-index 1 --submit-mode pbs --num-initial-conditions 4 --maximum-propagation-time 10 --time-step 0.5 --save-interval-steps 2 --device cpu
python scripts/select_md_frames.py --config configs/base.yaml --round-index 1 --max-new-points 5
```

## 4. 关于 MD 这一步

- round 0 初始条件会全部标注
- 后续 MACE MD 使用当前 MACE delta 势传播
- 每条轨迹一旦第一次超过不确定度阈值，就在那个点停下
- 只有这些返回点才进入下一轮筛选与标注

## 5. 当前默认参数

在 `configs/base.yaml` 里，默认设置是：

- `initial_conditions_initial: 200`
- `initial_conditions_per_round: 100`
- `trajectory_mode: bidirectional`
- `maximum_propagation_time: 150`
- `time_step: 0.5`
- `save_interval_steps: 5`
- `ensemble: NVE`
- `max_new_points_per_round: 100`
